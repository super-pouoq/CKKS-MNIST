## 阅读前必读：全局心智模型

一条密文在 Phantom 里长这样（CKKS）：

- **多项式环** R_q = Z_q[X]/(X^N+1)，N = `poly_modulus_degree`（2 的幂，如 2^14）。
- **RNS 表示**：大模数 q = ∏ q_i 被拆成若干 ~40/60-bit 的素数 q_i（NTT-friendly），
  每个多项式都以“在各 q_i 下的余数”这一组并行的小多项式存储 → 天然适合 GPU 并行。
- **NTT 域**：为了让多项式乘法变成逐点乘，系数大多以 NTT（数论变换）形式存放。
- **层级（level / chain_index）**：模数链 q_L ⊃ ... ⊃ q_0，每次 rescale/mod-switch 砍掉一个 q_i，
  对应 `chain_index` 递减；`ContextData` 为每一层预计算好所有常量。
- **密文** = 2 个（乘法后暂时 3 个）R_q 上的多项式 (c0, c1[, c2])。

GPU 视角：几乎所有数据是 `uint64_t*` 的扁平数组，按 `[modulus_idx][coeff_idx]` 排布，
kernel 用 `blockDimGlb=128` 的线程网格逐系数并行。

1. 想“快速理解一次 CKKS 推理”：顺读第 1→7 章即可。
2. 想“看懂某个 kernel 的 GPU 细节”：先读对应章节，再跳附录 A/B 看数学原语。
3. 每章结尾有“函数速查表”，可当 API 参考。
